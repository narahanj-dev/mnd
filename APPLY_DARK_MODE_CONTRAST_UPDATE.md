# 다크모드 가독성 개선 적용 안내

## 수정 파일
- `app/globals.css`
- `app/login/page.tsx`

## 주요 변경사항
- 로그인 공지사항을 라이트·다크모드 전용 고대비 디자인으로 변경
- 적색·청색·녹색·황색 상태 배지와 오류/성공 안내 색상 보정
- 달력 위험일 및 마우스 오버 색상 보정
- 카드, 표 구분선, 입력창, 비활성 입력창·버튼, 자동완성 입력 배경 보정
- 수동 테마 방식(`data-theme`)에서도 Tailwind `dark:` 스타일이 적용되도록 설정
- 브라우저 기본 날짜/시간 입력 UI도 현재 테마의 색상 체계를 사용하도록 개선

## 적용 방법
기존 프로젝트 폴더에 압축파일 내용을 덮어쓴 뒤 GitHub에 푸시하면 됩니다.
Supabase SQL 작업은 필요하지 않습니다.
