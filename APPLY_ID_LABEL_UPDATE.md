# 회원가입·사용자 관리 군번 문구를 아이디로 변경

## 변경 내용

- 회원가입 신청 화면의 `희망 군번`을 `희망 아이디`로 변경
- 관리자 가입 신청 화면의 `희망 군번`, `발급할 군번`을 아이디 문구로 변경
- 사용자 관리 화면의 검색, 목록, 생성, 수정, 삭제 확인 문구를 아이디로 변경
- 해당 기능에서 표시되는 중복 및 입력 오류 안내 문구를 아이디로 변경

## 적용

이번 변경은 화면 표시 문구만 수정하므로 Supabase SQL 실행은 필요하지 않습니다.
소스 파일을 덮어쓴 뒤 빌드하고 GitHub에 푸시하면 됩니다.
