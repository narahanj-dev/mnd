# 로그인 공지사항 수정 적용 안내

## 수정 내용
로그인 화면의 공지사항을 다음 내용으로 변경했습니다.

1. 회원가입 시 희망 아이디로 군번 및 숫자로만 이루어진 아이디는 사용할 수 없습니다.
2. 관리자가 비밀번호를 초기화하면 임시 비밀번호는 `mnd890701!`로 변경되며, 로그인 후 반드시 새 비밀번호로 변경해야 합니다.

## 적용 방법
1. 수정 압축파일을 풉니다.
2. 압축파일 안의 전체 파일을 기존 프로젝트 폴더에 덮어씁니다.
3. 기존 `.env.local` 파일은 유지합니다.
4. PowerShell에서 아래 명령어를 실행합니다.

```powershell
cd D:\leave-calendar
npm install
npm run build
git add .
git commit -m "로그인 공지사항 수정"
git push
```

이번 수정은 Supabase SQL 실행이 필요하지 않습니다.
