# 로그인 화면 심플화 적용 안내

## 변경 내용

- 로그인 화면의 `관리자 설정` 버튼과 관리자 전용 로그인 팝업 제거
- 로그인 화면의 공지사항 영역 전체 제거
- `부서 공동 연차달력` 문구 제거
- `휴가·외출·일정·기념일 통합 관리` 문구 제거
- 로그인 후 관리자가 사용하는 기존 관리자 설정 메뉴는 유지
- 아이디, 비밀번호, 로그인 버튼과 달력 아이콘만 표시
- 세션 만료 및 회원가입 종료 안내는 해당 상황에서만 표시

## 수정된 파일

1. `app/login/page.tsx`
2. `components/auth/LoginForm.tsx`

## 적용 방법

전체 압축파일을 기존 프로젝트에 덮어쓴 뒤 GitHub에 커밋·푸시하면 됩니다.
이번 변경에는 Supabase SQL 실행이나 환경변수 추가가 필요하지 않습니다.
