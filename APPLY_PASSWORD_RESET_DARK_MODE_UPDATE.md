# 비밀번호 초기화 및 다크모드 적용 안내

## 반영 내용

1. 사용자 관리에서 비밀번호 초기화 시 임시 비밀번호가 `mnd890701!`로 설정됩니다.
2. 초기화된 사용자는 로그인 직후 새 비밀번호로 변경해야 합니다.
3. 모든 화면 오른쪽 아래에 다크모드/라이트모드 전환 버튼이 표시됩니다.
4. 선택한 화면 모드는 해당 브라우저에 저장되어 다음 접속에도 유지됩니다.

## 적용 방법

이번 수정은 데이터베이스 구조 변경이 없어 Supabase SQL을 실행할 필요가 없습니다.

1. 기존 프로젝트의 `.env.local` 파일을 별도로 백업합니다.
2. 수정 압축파일을 해제합니다.
3. 압축을 푼 파일 전체를 기존 프로젝트 폴더에 덮어씁니다.
4. 기존 `.env.local`은 그대로 유지합니다.
5. 프로젝트 폴더에서 아래 명령어를 실행합니다.

```powershell
npm install
npm run build
npm run dev
```

6. 사용자 관리에서 비밀번호 초기화를 시험합니다.
7. 임시 비밀번호 `mnd890701!`로 로그인했을 때 비밀번호 변경 화면이 표시되는지 확인합니다.
8. 화면 오른쪽 아래 다크모드 버튼을 눌러 로그인 화면과 달력 화면이 모두 변경되는지 확인합니다.
